# Word Counter (150 Word Limit)

This project is a simple **Word Counter web application** built using **HTML, CSS, and JavaScript**.  
It allows users to type text and automatically counts the number of words entered, with a **maximum limit of 150 words**.

When the word limit is exceeded, the input resets and the counter starts again from **0**, as per the experiment requirement.

This experiment demonstrates the implementation of a Word Counter web application using basic web technologies.

The application allows the user to enter text into a text area and displays the live word count.
A maximum limit of 150 words is enforced. When the user exceeds this limit, the text input is automatically cleared and the word count resets to 0, ensuring controlled input as per the requirement.

The interface is styled with a dark-themed counter bar, closely resembling a real-time dashboard-style component.
